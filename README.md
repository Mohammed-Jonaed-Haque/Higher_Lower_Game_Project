# 🎮 Higher Lower Game (Python)
## Overview

This project is a command-line Higher Lower game built in Python, where players guess which of the two accounts has more followers using randomly selected data.

## Features

- Compare two random accounts

- Guess which one has more followers

- Score tracking system
- Continuous gameplay until wrong answer
- Uses real-style data (name, description, country)
- Simple and interactive console UI
##  Technologies Used
- Python
- Random module
- Dictionaries & Lists
- Functions
- Loops and Conditional Statements
## How It Works
- The game randomly selects two accounts from a dataset.
- The player is shown both accounts (A vs B).
- The player guesses which account has more followers.
- If correct → score increases and game continues.
- If wrong → game ends and final score is shown.


**Example:**
````
Compare A: Instagram, a Social media platform, from USA  
VS  
Compare B: Cristiano Ronaldo, a Footballer, from Portugal  

Who has more followers? a/b: b  

You're right! Score: 1
````
